Comment saisir les scores ?
Sélectionnez le mois et le projet, puis remplissez les colonnes pour chaque agent. Le score est calculé automatiquement. Cliquez sur 💾 Enregistrer quand vous avez terminé.
📐 Voir le détail du calcul du score
1. Score de base (sur 100%)
Performance × 40% + Qualité × 30% + Présence × 20% + Engagement × 10%

📋 Performance selon le type de projet :
Standard — saisie manuelle directe.
Télévente — calculée auto : CA réalisé ÷ Objectif individuel × 100.
BPO — calculée auto : Volume réalisé ÷ Objectif individuel × 100.

2. Engagement — 3 critères cochés (briefs, remontées, comportement) : 3/3 = 10%, 2/3 = 7%, 1/3 = 3%, 0/3 = 0%.

3. Pénalité avertissement — chaque avertissement retire 10% du score total. À partir du 3ème avertissement, le score tombe à 0%.

4. Rangs attribués
🏆 Rang S — top 5 du mois ET score ≥ 95%  ·  🥇 Rang A — ≥ 90%  ·  ⭐ Rang B — ≥ 75%  ·  ✅ Rang C — ≥ 60%  ·  ⚠️ Rang D — < 60%

📊 Scoring
🧮
Formule de calcul du score final
Score = Perf×40% + Qualité×30% + Présence×20% + Engagement×10%
Chaque avertissement déduit des points. Rang S = top 5 du mois et ≥95%, A ≥ 90%, B ≥ 75%, C ≥ 60%, D < 60%.
🏆
Bonus Rang A
Les 5 meilleurs agents du mois reçoivent automatiquement un badge Top 5. Les agents Rang A 3 mois consécutifs obtiennent le badge Régularité.
⚠️
Alertes Rang D
Tout agent passant en Rang D apparaît immédiatement dans la section Alertes du tableau de bord. Agissez rapidement : accompagnement ou plan d'action.

❓ Questions fréquentes
Comment ajouter un nouvel agent ? ›
Allez dans Agents → cliquez sur + Ajouter un agent. Renseignez son nom, prénom, projet et date d'entrée. L'agent apparaîtra immédiatement dans la liste de saisie des scores.
Comment saisir les scores du mois ? ›
Allez dans Administration → Mois & Scores. Sélectionnez le mois et le projet. Remplissez les colonnes pour chaque agent : Performance (objectif réalisé en %), Qualité (%), Présence (%) et cochez les critères d'Engagement. Cliquez sur Enregistrer.
Qui peut modifier quels champs du scoring ? ›
Admin / Respo : tous les champs.
Manager : Performance, Présence, Engagement et Avertissements.
Qualticien : Qualité uniquement.
Chef de projet : lecture seule sur le scoring.
Comment fonctionne le calcul des rangs ? ›
1. Score de base (sur 100%)
Performance × 40% + Qualité × 30% + Présence × 20% + Engagement × 10%

2. Engagement — 3 critères cochés (briefs, remontées, comportement) : 3/3 = 10%, 2/3 = 7%, 1/3 = 3%, 0/3 = 0%.

3. Pénalité avertissement — chaque avertissement retire 10% du score total (avert. 1 = -10%, avert. 2 = -20%). À partir du 3ème avertissement, le score tombe à 0%.

4. Rangs attribués
🏆 Rang S — top 5 du mois ET score ≥ 95%
🥇 Rang A — score ≥ 90%
⭐ Rang B — score ≥ 75%
✅ Rang C — score ≥ 60%
⚠️ Rang D — score < 60% (déclenche une alerte)

Exemple : Perf 90%, Qualité 85%, Présence 100%, Engagement 3/3, aucun avertissement → (90×0,4)+(85×0,3)+(100×0,2)+10 = 36+25,5+20+10 = 91,5% → Rang A.
Comment fonctionne le scoring Standard / Télévente / BPO ? ›
Chaque projet a un type d'activité qui détermine comment la Performance est calculée. La Qualité, Présence et Engagement fonctionnent toujours pareil, quel que soit le type.

📋 Standard — Le % de Performance est saisi directement à la main par le Manager, comme pour la majorité des projets (Japhy, Walter Santé classique, etc.).

💰 Télévente (Edumiam, Walter Santé CA, Walter Alternance...) — On saisit le CA réalisé en € par l'agent. La Performance se calcule automatiquement :
Performance % = CA réalisé ÷ Objectif individuel de l'agent × 100
Exemple : objectif 5 000€, réalisé 5 500€ → 110% de Performance.

📞 BPO (Patimovie, Japhy BPO...) — On saisit le volume réalisé (RDV pris, flux traités...) par l'agent. Même calcul automatique, mais en nombre plutôt qu'en € :
Performance % = Volume réalisé ÷ Objectif individuel de l'agent × 100
Exemple : objectif 15 RDV, réalisé 12 RDV → 80% de Performance.

Deux niveaux d'objectif — chaque projet a un objectif global (vision macro, ex: 50 000€ pour tout le projet) ET chaque agent a son objectif individuel (ex: 5 000€). Le calcul de la Performance utilise toujours l'objectif individuel.

Le type d'activité se choisit à la création du projet, ou se modifie ensuite via le bouton "Ops" sur la page Projets.
Comment créer un nouveau compte utilisateur ? ›
Réservé aux Admins. Allez dans Administration → Comptes. Cliquez sur + Nouveau compte, choisissez le rôle et définissez les identifiants. Le compte est actif immédiatement.
Comment exporter un rapport PDF ? ›
Cliquez sur 📄 Export PDF dans le header (desktop) ou dans le menu Plus (mobile). Choisissez le périmètre (tous les projets ou un seul), le mois et les sections à inclure. Cliquez sur Générer.
Les données sont-elles sauvegardées automatiquement ? ›
Oui. Après chaque clic sur Enregistrer, les données sont synchronisées en temps réel sur Firebase Firestore. Le badge ✅ Synchronisé en haut à droite confirme la sauvegarde. Une connexion internet est nécessaire.
Comment planifier une évaluation dans le calendrier ? ›
Allez dans Calendrier & Deadlines. Cliquez sur un jour ou sur + Ajouter. Renseignez le titre, la catégorie (Évaluation, Formation, Réunion…), les rôles destinataires et une date limite si nécessaire.
Que signifie le badge "Régularité" ? ›
Ce badge est attribué automatiquement aux agents qui maintiennent le Rang A pendant 3 mois consécutifs. Il apparaît sur la fiche de l'agent et dans le classement Top 5.
Puis-je voir l'historique d'un agent sur plusieurs mois ? ›
Oui. Dans Agents, cliquez sur un agent pour ouvrir sa fiche. Vous y trouverez un graphique d'évolution mensuelle, l'historique des rangs, les badges obtenus et les commentaires.
